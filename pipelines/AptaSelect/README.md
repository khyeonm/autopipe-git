# AptaSelect

Identifies high-frequency aptamer candidate sequences from paired-end FASTQ files generated by SELEX experiments.

## Pipeline overview

| Stage | Description |
|-------|-------------|
| 1. Join | Merge paired-end reads via 3′→5′ overlap; resolve mismatches by Phred quality |
| 2. Selection | Retain reads containing `SEL_READ1` … `SEL_READ2`; extract the between region |
| 3. 1st Sort | Further filter by `S1_READ1` … `S1_READ2` with exact inter-pattern length (`S1_LENGTH`) |
| 4. 2nd Sort | Final filter by `S2_READ1` … `S2_READ2` with exact inter-pattern length (`S2_LENGTH`) |
| 5. Aggregation | Sum counts across parallel chunks; output top-N ranked aptamer sequences |

## Inputs

| File | Description |
|------|-------------|
| `fastq1` | Paired-end read 1 (`.fastq` or `.fastq.gz`) |
| `fastq2` | Paired-end read 2 (`.fastq` or `.fastq.gz`) |

## Output

`/output/aptaselect_results.tsv` — tab-separated file with columns:

```
rank    sequence    count
1       ATCG...     4821
...
```

## Configuration (`config.yaml`)

| Parameter | Default | Description |
|-----------|---------|-------------|
| `min_overlap` | 6 | Minimum overlap length for read joining (bp) |
| `pct_diff` | 8.0 | Maximum mismatch % in overlap |
| `sel_read1/2` | see defaults | Broad selection pattern pair |
| `s1_read1/2` | see defaults | 1st sort pattern pair |
| `s1_length` | 40 | Required bp between 1st sort patterns |
| `s2_read1/2` | see defaults | 2nd sort pattern pair |
| `s2_length` | 20 | Required bp between 2nd sort patterns |
| `max_mismatches` | 1 | Max mismatches allowed in all pattern searches |
| `top_n` | 10 | Number of top candidates to output |
| `chunk_size` | 10000 | Reads per parallel chunk |
| `threads` | 4 | Worker threads |

## How to run

```bash
# 1. Build Docker image
docker build -t aptaselect .

# 2. Edit config.yaml to point to your FASTQ files and adjust parameters

# 3. Run
docker run --rm \
  -v /path/to/input:/input:ro \
  -v /path/to/output:/output \
  aptaselect \
  snakemake --cores 4 --nolock
```
